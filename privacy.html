Kebijakan Privasi

Dokumen ini menjelaskan bagaimana RekberWE.id mengumpulkan, menggunakan, menyimpan, dan melindungi data pengguna saat memakai layanan rekber kami.

1. Data yang kami kumpulkan

Kami dapat mengumpulkan data akun, data profil, data transaksi, pesan chat, lampiran file, data verifikasi identitas, serta informasi login dari penyedia OAuth seperti Google, Facebook, Discord, dan Telegram.

2. Penggunaan data

  • Memproses login dan autentikasi pengguna.
  • Menjalankan transaksi rekber antara pembeli dan penjual.
  • Menyimpan bukti transaksi, histori chat, dan status layanan.
  • Melakukan verifikasi identitas dan penanganan sengketa.
  • Meningkatkan keamanan sistem dan mencegah penyalahgunaan.

3. Penyimpanan dan keamanan

Data disimpan pada infrastruktur yang kami gunakan untuk operasional layanan, termasuk database aplikasi dan penyimpanan file pihak ketiga bila diperlukan. Kami berupaya menjaga keamanan data dengan kontrol akses admin dan pembatasan akses sesuai kebutuhan operasional.

4. Pembagian data

Kami tidak menjual data pribadi pengguna. Data hanya dapat diakses oleh admin RekberWE.id untuk kepentingan verifikasi, moderasi, penyelesaian sengketa, dan dukungan pengguna. Dalam kasus sengketa serius seperti dugaan hackback, admin dapat menggunakan data verifikasi sesuai kebijakan layanan dan ketentuan hukum yang berlaku.

5. Hak pengguna

Pengguna dapat meminta pembaruan data profil atau menghubungi admin untuk pertanyaan terkait data pribadi, perubahan data verifikasi, atau permintaan peninjauan data tertentu.

6. Layanan pihak ketiga

Kami dapat menggunakan layanan pihak ketiga seperti OAuth provider, penyimpanan file, dan infrastruktur hosting. Penggunaan layanan tersebut tetap mengikuti kebijakan privasi dari masing-masing penyedia.

7. Perubahan kebijakan

Kebijakan privasi ini dapat diperbarui sewaktu-waktu mengikuti perkembangan layanan. Versi terbaru akan selalu tersedia pada halaman ini.

8. Kontak

Jika ada pertanyaan tentang kebijakan privasi ini, silakan hubungi admin RekberWE.id melalui fitur live chat di website.
