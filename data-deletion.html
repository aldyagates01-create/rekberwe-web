Penghapusan Data Pengguna

Panduan ini menjelaskan cara pengguna meminta penghapusan data dari RekberWE.id.

Jika Anda ingin meminta penghapusan data akun atau data tertentu yang tersimpan di RekberWE.id, silakan hubungi admin melalui fitur live chat di website dan sertakan identitas akun yang digunakan saat login.

Langkah permintaan penghapusan data
  1. Login ke akun Anda di RekberWE.id, jika masih memiliki akses.
  2. Hubungi admin melalui live chat dan sampaikan bahwa Anda ingin menghapus data pengguna.
  3. Sebutkan data yang ingin dihapus, misalnya akun, profil, atau data verifikasi.
  4. Admin dapat meminta verifikasi tambahan untuk memastikan permintaan berasal dari pemilik akun yang sah.

Data yang mungkin tetap disimpan

  • Data transaksi yang masih aktif atau sedang sengketa.
  • Data yang wajib disimpan sementara untuk alasan keamanan, anti-penipuan, atau kepatuhan operasional.
  • Catatan administratif yang diperlukan untuk audit internal layanan.

Waktu penanganan

Permintaan penghapusan data akan ditinjau oleh admin. Waktu penanganan dapat menyesuaikan kondisi transaksi, status sengketa, dan kebutuhan verifikasi identitas pemilik akun.
